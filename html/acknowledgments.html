Acknowledgment

Actimetre System makes use of the following hardware and software components.

Actimetre

Hardware

  • S2 mini and S3 mini boards by LOLIN
  • Based on the ESP32-S2 and ESP32-S3 chips manufactured by Espressif Systems
  • Sensor breakout board GY-521 based on MPU-6050 by Invensense, a TDK company

Software

  • Arduino IDE with Espressif's ESP32 Arduino Core

Actiserver

Hardware

  • amd64 based Ubuntu 22.04 system optionally with RTL88x2AU/BU WiFi chip on USB, with drivers from morrownr
    or
  • ARM based Debian Bullseye or Ubuntu 22.04 system with on-board WiFi support.
    Tested on Raspberry Pi 3, Pi 4, Compute Module 4; NanoPi Neo3 with USB WiFi dongle; OrangePi Zero2.

Software

  • OpenJDK-17 running Kotlin code written on IntelliJ IDEA
  • On Raspberry hardware, the Embedded Linux JVM Debugger plug-in by asebak was used

Acticentral

Acticentral is really independent of hardware. It was tested on various single-board computers, Linux PCs, and AWS Lightsail.

Software

  • Apache2 for HTTP
  • Python3 for the main script
  • Matplotlib for graphs
  • Yattag for HTML generation with Python

Generic software

  • GNU Emacs for code editing
  • KiTTY as SSH client of choice
  • Github for version management
